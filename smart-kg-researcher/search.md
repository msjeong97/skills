# Phase 1: 지식 탐색

사용자 질문에서 핵심 키워드를 1~3개 추출한 뒤 아래 두 검색을 수행합니다.

---

## 1. MCP 그래프 검색

MCP `search_nodes` 도구로 각 키워드를 검색합니다.
- 결과: 관련 노드(엔티티) 목록과 연결된 엣지(관계) 정보

## 2. 파일 검색

Grep 도구로 `~/knowledge/**/*.md` 에서 키워드를 검색합니다.
- 키워드가 여러 개면 각각 검색 후 결과 파일 목록을 합집합으로 합칩니다.
- 결과: 키워드가 포함된 `.md` 파일 경로 목록

---

## 충분성 판단 (Sufficiency Rule)

| 조건 | 결과 | 다음 단계 |
|---|---|---|
| MCP 결과 ≥1 AND Grep 결과 ≥1 | 충분 | answer.md (Phase 2) |
| 둘 중 하나만 결과 있음 | 부분 | answer.md (Phase 2, 부분 커버리지 명시) |
| 둘 다 결과 없음 | 불충분 | research.md (Phase 3) |

충분/부분 판단 시: 매칭된 파일들을 Read 도구로 읽어 내용을 확인한 후 answer.md로 넘어갑니다.

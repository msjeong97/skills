# Phase 2: 답변 및 그래프 통찰

## 답변 형식

모든 답변은 아래 구조를 따릅니다:

### 1. 출처 명시 (첫 줄)

충분한 경우:
> "내부 지식 그래프와 리서치 파일 기반 답변입니다."

부분 커버리지인 경우:
> "내부 지식 그래프와 리서치 파일 기반 답변입니다. (일부 정보만 존재합니다)"

### 2. 본문

검색된 `.md` 파일 내용과 MCP 그래프 노드 정보를 종합하여 답변합니다.

### 3. 그래프 통찰 (관련 엣지가 있을 때)

MCP 엣지 정보를 활용하여 연결 관계를 표현합니다:
> "이 주제는 [엔티티 A]와 [관계 유형]으로 연결되어 있습니다."

예시:
> "Apache Kafka는 Kafka Connect와 data_pipeline 관계로 연결되어 있으며, Apache Iceberg와 storage_sink 관계가 있습니다."

엣지가 없으면 이 섹션은 생략합니다.

### 4. 참조 파일

답변에 사용된 `.md` 파일 경로를 마지막에 나열합니다:
```
참조: ~/knowledge/research/paimon/01-concepts-overview.md
```

# 저평가 지수 산출 방법론

## 총점: 100점 (저평가 지수) + 7점 (상승 신호) = 107점 만점

## 사전 자격 필터 (Pass/Fail)
- ROE > 10%
- 최근 4분기 각각 영업이익 흑자
- D/E < 100 (yfinance 퍼센트 단위, 즉 실질 D/E < 1.0)

## A. 밸류에이션 (25점)
- PE vs 섹터 중위수: 하위 30% = 10점, 하위 50% = 5점
- FCF Yield: >5% = 10점, 3~5% = 6점, 1~3% = 3점, <1% = 0점
- PEG: <1.0 = 5점, 1.0~1.5 = 3점, 결측/기타 = 0점

## B. 수익성 품질 (20점)
- ROE: >20% = 10점, 15~20% = 7점, 10~15% = 4점
- 영업이익률(섹터 상위%): 상위 25% = 10점, 상위 50% = 5점

## C. 단기 기술적 반전 신호 (55점)
- 52주 저점 대비 위치: 5~20% 구간 = 15점, 20~35% = 7점
- RSI(14): 30~45 = 12점, 45~55 = 6점
- MACD(12,26,9) 골든크로스: 최근 5거래일 내 = 12점
- 볼린저밴드 %B 반등: 최근 10일 내 <0.2 터치 후 현재 >0.3 = 8점
- 가격+거래량 동반 상승: 최근 5일 내 +1%↑ + 거래량 20일평균 1.5배↑ = 8점

## D. 상승 신호 점수 (7점, AI 웹서치)
- 애널리스트 목표가 상향 (최근 4주): 3점
- 실적 발표 임박 2주 이내: 2점
- 내부자/기관 매수: 2점
